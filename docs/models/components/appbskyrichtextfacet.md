# AppBskyRichtextFacet

Annotation of a sub-string within rich text.

## Example Usage

```typescript
import { AppBskyRichtextFacet } from "bluesky/models/components";

let value: AppBskyRichtextFacet = {
  index: {
    byteStart: 919783,
    byteEnd: 36033,
  },
  features: [
    {
      did: "<id>",
    },
  ],
};
```
