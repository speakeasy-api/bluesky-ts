# @speakeasy-sdks/bluesky Examples

This directory contains example scripts demonstrating how to use the @speakeasy-sdks/bluesky SDK.

## Prerequisites

- Node.js (v18 or higher)
- npm

## Setup

1. Copy `.env.template` to `.env`:
   ```bash
   cp .env.template .env
   ```

2. Edit `.env` and add your actual credentials (API keys, tokens, etc.)

## Running the Examples

To run an example file from the examples directory:

```bash
npm run build && npx tsx example.ts
```
